# agefans.tv，AGE动漫追番扩展
本人是个番剧爱好者，Agefans是我比较常用的网站，可惜网站缺少追番和历史功能，有时候感觉不是很方便你，所以便萌生为Agefans开发扩展的想法。

## 下载
见[下载页面](https://github.com/leeggco/agefans_ex/releases)

## 功能
1、扩展用户注册登录

2、追番列表

3、历史列表

## 使用
1、下载并解压扩展文件

2、打开浏览器扩展页面（可直接在地址栏输入：chrome://extensions/）

3、打开开发者模式

4、加载已解压的扩展程序
